# 宠物管理系统简介 java1040

## 项目概述

宠物管理系统是一款基于前后端分离架构的宠物信息管理应用。系统旨在为用户提供宠物领养、救助信息发布、论坛交流等功能，同时为管理员提供全面的管理工具，以便高效管理用户、宠物信息以及相关活动。

## 技术栈

- **前端技术**：Vue.js、ElementUI、ECharts
- **后端技术**：Spring Boot、MyBatis-Plus
- **数据库**：MySQL

## 功能模块

### 系统角色

- 用户
- 管理员

#### 用户功能

- **首页**：展示公告列表、宠物科普、流浪宠物介绍、热门活动
- 注册、登录
- 宠物领养
- 宠物救助
- 丢失宠物查看
- 宠物论坛

#### 管理员功能

- 用户管理
- 申请领养管理
- 评论管理
- 流浪动物救助
- 动物走失管理
- 救助站管理
- 帖子管理
- 捐赠管理
- 公告管理
- 科普文章管理
- 活动管理

## 运行环境
